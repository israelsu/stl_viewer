# STL Viewer

A 3D STL file viewer built with Three.js.

## Features

- Upload and view STL files in 3D
- Orange theme UI
- Centered coordinate system (UCS)
- Units displayed in mm
- Single file upload (replaces previous file)
- Orbit controls for navigation

## How to Run

1. Open `index.html` in a web browser.
2. Click "Upload STL File" to select an STL file.
3. Use mouse to rotate, zoom, and pan the view.

## Requirements

- Modern web browser with WebGL support

## Notes

- STL files are assumed to be in mm units.
- The coordinate system origin is at the center of the view.
